*[MICOS]: Metagenomic Intelligence and Comprehensive Omics Suite - 宏基因组智能与综合组学套件
*[宏基因组]: Metagenomics - 研究环境样本中全部基因组集合的学科
*[CLI]: Command Line Interface - 命令行界面
*[API]: Application Programming Interface - 应用程序编程接口
*[QC]: Quality Control - 质量控制
*[FASTQ]: 一种存储生物序列（通常是核苷酸序列）及其对应质量分数的文本格式
*[BIOM]: Biological Observation Matrix - 生物观察矩阵，用于存储微生物组数据的格式
*[OTU]: Operational Taxonomic Unit - 操作分类单元
*[ASV]: Amplicon Sequence Variant - 扩增子序列变体
*[PCoA]: Principal Coordinate Analysis - 主坐标分析
*[NMDS]: Non-metric Multidimensional Scaling - 非度量多维尺度分析
*[PERMANOVA]: Permutational Multivariate Analysis of Variance - 置换多元方差分析
*[CPM]: Counts Per Million - 每百万计数
*[TPM]: Transcripts Per Million - 每百万转录本
*[FDR]: False Discovery Rate - 错误发现率
*[LCA]: Lowest Common Ancestor - 最低共同祖先
*[k-mer]: 长度为 k 的核酸序列子串
